HOW TO USE THE DECRYPTION TOOL
==============================

1. Open this folder in your terminal:
   cd decryption_tool

2. Run the script:
   python decrypt_viewer.py

3. When asked, drag and drop your encrypted backup file (e.g., parthborse07_backup_encrypted_...) into the window.

4. Enter your password.

5. A new decrypted file will be created in the same folder as your backup.
